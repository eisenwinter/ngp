NGP

ngp generiert aus dem QR code des Impfzertifkates einen kleinen Zettel, welcher den QR Code und die darin enthaltenen Daten beinhaltet. Dieser ist kaum größer als eine Euro-Bank Note und kann komfortabel im Geldbeutel mitgeführt werden.
Dieses Projekt ist für all jene die entweder interessiert was sich hinter dem QR an Daten verbirgt oder diesen praktischen Zettel möchten.

Achtung!

Prinzipiell beinhaltet der QR Code sensible Gesundheitsdaten und sollte nicht einfach im Internet hergegeben werden.

Die hochgeladenen Daten werden nicht gespeichert und ausschließlich einmal gelesen, um den "ngp" zu generieren. Sollte trotzdem Misstrauen bestehen kann die Anwendung auch lokal geladen werden, um die Datenhoheit zu behalten. Mehr dazu in der Sektion Open Source.

FAQ

Frequently Asked Questions

Warum überhaupt?

Einerseits war's Interesse, was tatsächlich in dem QR Code steht, andererseits hat mich gestört, wie groß das ausgestellte PDF war (A4). Auch wenn es heißt, der QR Code alleine ist genug, werden stets nur die tabellarischen Daten geprüft - also sollten diese auch ersichtlich sein.

Funktioniert das mit allen europäischen Impfzertifikaten?

Theoretisch ja, getestet ist es aber nur mit Österreichischen.

Kann die Anwendung die Signatur prüfen?

Nein, das kann nur der Aussteller (in Österreich das Gesundheitsministerium) und ist nur durch den offiziellen Scanner geprüft werden.

Ist das ein offizielles Dokument?

Nein. Das ganze ist ein Hobby Projekt, es ist kein offizielles Dokument. Das erzeuge Dokumente besteht lediglich aus den Daten des offiziellen QR Code des Impfzertifkates.

Wird das Dokument akzeptiert?

Prinzipiell sollte eigentlich der QR Code ausreichen, dieser wird unmodifiziert auf das Dokument gedruckt und verliert damit so nicht seine Gültigkeit.
Ob es in der Realität in Gasthäusern oder Sportstätten akzeptiert wird, kann ich nicht garantieren.

Kann ich hiermit ein Dokument mit falschen Daten generieren?

Nein. Selbst wenn Sie einen gefälschten QR Code erzeugen könnten, so würde dies beim Scannen mit dem offiziellen Scanner erkannt werden.

Was bedeutet die lange Zeichenkette auf dem Dokument?

Das ist die eindeutige Kennung des Zertifikates.
